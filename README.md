# 🖨️ Vprint – Anonymous Campus Printing Tool

**Vprint** is a lightweight web app that solves the hassle of printing documents on shared/public PCs without compromising user privacy.  
Built specifically for student environments where WhatsApp was previously used for transferring files to printing stations.

---

## 🚩 Problem

On our campus, students had to:

- Log in to **WhatsApp Web** on a public PC  
- Send documents to themselves  
- Download them manually to print

This caused:
- 😰 **Privacy issues** (chats exposed on screen)
- 🤐 **Discomfort** sharing personal numbers
- 🕒 **Queue delays** due to repeated manual effort

---

## 💡 Solution

**Vprint** allows students to:

1. Upload documents using a **guest code**
2. Retrieve them later using that same code
3. Download & print files without logging into anything

---

## ✨ Features

- 🔐 **Guest code-based upload and download**
- 🕐 **Files auto-delete after 7 days**
- 📊 **Real-time "print" statistics** (downloads)
- 📁 Supports `.pdf`, `.doc`, `.docx`, `.png`, `.jpg`, `.jpeg`, `.txt`
- ⚡ Simple, fast, and 100% anonymous — no login required

---

## 🧰 Tech Stack

- **Frontend:** Next.js, TailwindCSS  
- **Backend/Storage:** Supabase  
- **Deployment:** Vercel  
- **Data Cleanup:** Supabase policies for auto-expiry
